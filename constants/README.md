# Constants Directory

Application-wide constants and static data.

## Files

- `branches.ts` - KTU branch definitions
- `semesters.ts` - Semester configuration  
- `grades.ts` - Grade point mappings (deprecated, use features/sgpa/constants)
- `routes.ts` - Route constants

## Usage

```ts
import { BRANCHES, SEMESTERS } from '@/constants';
```
